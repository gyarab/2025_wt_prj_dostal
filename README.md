# Webová aplikace

## Instalace a spuštění

### 1. Vytvoření venv

**Windows:**

```bash
py -3 -m venv venv
source venv/Scripts/activate
pip install -r requirements.txt
```

**MacOS / Linux**

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 2. Spuštění aplikace

**Windows:**

```bash
py -3 ./prj/manage.py runserver
```

**MacOS / Linux:**

```bash
python3 ./prj/manage.py runserver
```
